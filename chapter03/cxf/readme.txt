Mule SOAP Demos
---------------------------------

You must first build this project with: mvn clean install

Start this example with the script that is relevant for your OS.

Point your SOAP client at the following WSDL to test the examples:

http://localhost:9781/greeting?wsdl

SoapUI is a good SOAP client if you don't have one.  It can be downloaded
from here:

http://www.soapui.org/
